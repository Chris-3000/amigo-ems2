Sprite Editor to Arduino TFT Code

Create a pixel sprite, preview it, and export it as Arduino code for TFT displays. This version exports a 16-bit RGB565 array that works well with libraries like Adafruit_GFX using drawRGBBitmap().

Sprite Setup

This app still exports full RGB565 pixel arrays. Transparent mode only adds a note in the output.

Editor

Left click or drag to draw. The editor scales automatically so you can work on small sprites easily.

Preview and Arduino Output

Example use: tft.drawRGBBitmap(x, y, mySprite, WIDTH, HEIGHT);
